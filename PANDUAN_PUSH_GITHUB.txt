PANDUAN PUSH KE GITHUB

Saya sudah menyiapkan file project yang berisi update tugas:
- Tabel tambahan: platforms
- CRUD platform lengkap
- Watchlist terhubung dengan platform melalui platform_id

Karena akses GitHub membutuhkan login/token milik pemilik akun, push harus dilakukan dari laptop Anda.

Langkah push:
1. Buka folder project di VS Code atau terminal.
2. Jalankan perintah berikut:

   git add .
   git commit -m "Tambah CRUD platform streaming"
   git push origin main

Jika folder hasil extract tidak memiliki folder .git, gunakan cara berikut:

   git init
   git branch -M main
   git remote add origin https://github.com/Jauzaaaaaaaaaaaaaaa/simple-watchlist-website.git
   git add .
   git commit -m "Tambah CRUD platform streaming"
   git push -u origin main

Catatan:
- Pastikan sudah login GitHub di terminal/VS Code.
- Import ulang database.sql di phpMyAdmin agar tabel platforms dan relasi platform_id dibuat.
